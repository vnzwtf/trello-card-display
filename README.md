# trello-card-display
display trello cards with the related members on a stylish website

## Features

- Display Trello Cards
- Responsive Design
- Filter Members that own Cards

## Installation

```bash
    1. Download the Files
    2. Upload in on your Device
    3. Rename example.env to .env
    4. Edit the .env
    5. npm run build
    6. npm run start
```
